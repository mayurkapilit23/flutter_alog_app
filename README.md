# Flutter Assignment Project

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/project-name.git
```

### Navigate to Project

```bash
cd project-name
```

### Install Dependencies

```bash
flutter pub get
```

### Run Application

```bash
flutter run
```

---

## 📦 Build APK

```bash
flutter build apk --release
```

APK location:

```bash
build/app/outputs/flutter-apk/app-release.apk
```
