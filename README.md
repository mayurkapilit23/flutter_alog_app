# Flutter Task - Algorithm Task

## Task:
Make an app that shows a grid of numbers from 1 to 100. Then, depending on a rule we
choose, some numbers will be shown in a different way.

## What You Do:
- **Setting up the App:** First, you need to create an app layout that displays a grid of
  numbers. Imagine a big square where we put numbers from 1 to 100, like a big list but
  organized in rows and columns.

- **Finding the Rule:** Next, you'll need to come up with a rule or pattern to highlight certain
  numbers in the grid. This could be anything from highlighting all the even numbers to
  finding prime numbers or Fibonacci sequence numbers.

### Required Rules:
1. Odd Numbers
2. Even Numbers
3. Prime Numbers
4. Fibonacci sequence numbers

- **Highlighting Numbers:** Once you've decided on the pattern, you'll write code to identify
  the numbers that match that pattern. When the app runs, it will go through each number
  in the grid and check if it fits the rule. If it does, the app will highlight that number using a
  different color or style.

- **Changing the Rule:** The app should be able to change the highlighting pattern
  dynamically. This means that users can choose different rules to highlight numbers, such
  as switching from prime numbers to Fibonacci sequence numbers. When the user
  selects a new pattern, the app should update the grid to reflect the changes efficiently.

## Tips:
- **Efficient Algorithms:** Think about how to efficiently find numbers that match the
  pattern. For example, finding prime numbers or Fibonacci sequence numbers might
  require different algorithms.

- **Making it Smooth:** When people change the rule, the app should be quick and smooth.
  Nobody likes waiting for things to happen on their phone.
